# Software Metrics

## Delivery Metrics
- Deployment frequency.
- Change lead time.
- Change failure rate.
- Mean time to recovery.

## Reliability Metrics
- Availability percentage and error budget burn.
- API error rate by endpoint and tenant.
- Incident count by severity and recovery duration.

## Performance Metrics
- Latency by percentile (p50, p95, p99).
- Throughput (requests per second).
- Queue depth and processing lag for async workloads.
- Cache hit ratio and database query latency.

## Security Metrics
- Failed authentication rate.
- Blocked abuse and rate-limit events.
- Time to remediate vulnerabilities.
- Secrets rotation compliance percentage.

## Quality Metrics
- Test pass rate and flaky test ratio.
- Code coverage in critical services.
- Static analysis and dependency health.
- Defect escape rate.

## AI and Data Metrics
- Model quality metrics by use case.
- Data drift and concept drift indicators.
- Recommendation acceptance and feedback quality.
- Cost per inference and model latency.
